# 💄 Cosmetic Chemical Awareness Project

## 📌 Project Overview
This project investigates the presence and frequency of harmful chemicals (such as parabens, phthalates, and formaldehyde) in various cosmetic products 
Using data analysis and machine learning, the study identifies hazardous formulations to enhance consumer awareness and support safer product choices.

---

## 🛠 Methodology & Technical Implementation
* **Data Acquisition:** Gathered ingredient lists from cosmetic databases and regulatory sources.
* **Data Preparation:** Standardized ingredient names and engineered a target feature `is_harmful` based on known hazardous substances.
* **Machine Learning:** Developed a **Logistic Regression** model to predict the presence of harmful chemicals using vectorized ingredient text.
* **Analytics:** Conducted statistical grouping by brand and category to identify high-risk product groups.

---

## 📊 Key Visualizations
The project includes several analytical charts:
* **Risk Distribution:** Proportion of harmful vs. non-harmful ingredients.
* **Brand & Category Analysis:** Identification of top 10 categories and brands associated with hazardous chemicals.
* **Predictive Performance:** Evaluation through confusion matrices and classification reports.
